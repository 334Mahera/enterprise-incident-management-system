# Enterprise Incident Management System

A corporate-style SAP Application Support simulation portal built using Flask.

## Key Features

- Role-based authentication (Admin / User)
- Incident ticket management
- SLA resolution time calculation
- Analytics dashboard with chart visualization
- Corporate navbar UI
- 3D animated login & signup pages
- Fully responsive design

## Tech Stack

- Python
- Flask
- SQLite
- HTML5
- CSS3
